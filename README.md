## 📚 Grade Tracker

A lightweight, browser-based grade tracking app built with pure HTML, CSS, and JavaScript.

# Forking This Project
Feel free to fork this repository and customize it for your own needs!

# What you can do after forking:
- Customize the UI (colors, layout, themes)
- Modify or expand subject templates
- Improve the PIN/login system
- Add cloud sync (Firebase, Supabase, etc.)
- Add charts or analytics
- Convert it into a full PWA (installable app)

# Project Structure
index.html   → Main app (UI + logic)
README.md    → Documentation
# This project is intentionally kept simple (single file) for easy editing and portability.

# Data Storage
- Uses "localStorage"
- Data stays in the user's browser
- No backend required

# Important Notes
- Clearing browser data will erase all saved grades
- PIN system is basic (not secure for sensitive data)
- No server-side storage
